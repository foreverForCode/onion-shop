# 常用方法


### jems.transformLink

> 转化 == content == 为链接

``` js
jems.transformLink = function(content){

    var reg = /==(\S*)==/g;

    var originData = content.match(reg);
    if(!originData){

        originData = [];
    }
    var linkData = originData.map(function(item){
        
        return item.replace(/==/g, '');
    
    })

    return {

        originData: originData,
        linkData : linkData
    }
}
```
